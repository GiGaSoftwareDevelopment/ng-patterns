# @ngpat/slick-carousel

This is a native Angular implementation of [https://github.com/kenwheeler/slick](https://github.com/kenwheeler/slick).

See demo at [https://ng-patterns.web.app/slick/carousel](https://ng-patterns.web.app/slick/carousel).


### Settings

"Planned" settings will be implimented sometime in the future ( PRs welcome, ahem ). "Implemented" indicates which features have been completed and released. This is a work in progress, please be patient and again, PRs welcome.
