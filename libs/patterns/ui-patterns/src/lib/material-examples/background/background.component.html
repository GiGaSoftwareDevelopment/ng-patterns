Override Material Dark Theme Background Configure a mixin to override Angular Material's background classes. Install API This is a scss mixin to override material's dark theme background configuration. The mixin is signature and parameter definitions are:
$theme-object Material theme object.
$palette Material color palette ( does not have to be the palette from theme object ).
$lightness Lightness or darkness adjustment percentage based on color palette. This is relative from 0, where 0 is no adjustment. Negative is darker, positive is lighter.
$saturation Saturation adjustment percentage based on color palette. This is relative from 0, where 0 is no adjustment. Negative is less saturation, positive is more saturation.
